# Jeremy Hummel · Portfolio

A retro-futuristic developer portfolio for Tools Engineer Jeremy Hummel. The site channels neon-coded 80s vibes while highlighting experience designing intuitive creation tooling.

## Getting started

Open `index.html` directly in your browser, or serve the site locally:

```bash
python3 -m http.server
```

Then visit [http://localhost:8000](http://localhost:8000).

## Structure

- `index.html` – Main single-page portfolio.
- `assets/css/style.css` – Styling inspired by cassette futurism aesthetics.

## Customization

Update the content in `index.html` to keep résumé details current. Adjust theme colors, animations, or layout via `assets/css/style.css`.
